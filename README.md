# Development and Contributing

## Development Setup & Start
At the project root:

__Step 1:__ Create a virtual environment  `.venv`.

__Step 2:__ Then run:
```
python setup_dev.py
```

__All done setting up!__

__Step 3:__ Start the application by running:
```
poetry run wave
```
